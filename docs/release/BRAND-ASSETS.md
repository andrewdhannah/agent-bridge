## Brand position

agent-bridge should feel:

- secure
- local-first
- audited
- precise
- non-authoritative
- human-validated
- technical but approachable

## Core visual language

- Blue-to-teal bridge motif
- Node/arc transport structure
- Signed intent / verification symbolism
- Dark release splash for security posture
- Clean geometric wordmark
- Minimal authority-safe UI language

## Primary tagline

```text
Secure browser-to-local agent handoff.
```

## Secondary line

```text
A trust bridge, not a control channel.
```

## Safety language

Use:

```text
signed intent
human-validated
audited
non-authoritative by design
read-only status
custody-linked review
```

Avoid:

```text
auto-approve
auto-execute
browser control
agent control
authorized by extension
permission from CSS
```
