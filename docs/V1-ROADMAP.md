# V1 Roadmap

The V1 goal is to establish a stable, human-verified pipeline for local agentic work.

## 🎯 Key Milestones

- [x] **Core Transport**: HTTP bridge and JSON queue.
- [x] **Human Gate**: Forced approval mechanism.
- [x] **MCP Toolset**: Standardized queue management tools.
- [ ] **Librarian Integration**: Automated routing and classification.
- [ ] **Evidence Verification**: Automated check of results against acceptance criteria.
- [ ] **Git Integration**: Automatic commit of completed work packets.

## 📦 V1 Definition of Done
V1 is complete when a prompt can flow from a web AI to a local git commit, with every step audited by The Librarian and approved by a human.
