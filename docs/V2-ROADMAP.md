# V2 Roadmap

V2 focuses on scaling the bridge from a single-agent pipeline to a multi-agent orchestration system.

## 🚀 Future Directions

### 1. Advanced Routing
- **Dynamic Dispatch**: Routing packets to different worker agents based on the Librarian's classification.
- **Parallel Execution**: Allowing multiple approved packets to be worked on simultaneously.

### 2. Secure Post-back
- **Controlled Feedback**: Implementing a secure, human-approved channel to send results back to the web AI.
- **Context Injection**: Feeding the result of local work back into the web AI's context window.

### 3. Enhanced Governance
- **Policy-Based Approval**: Auto-approving low-risk tasks based on a predefined policy.
- **Multi-Sig Approval**: Requiring multiple human approvals for high-risk operations.

### 4. UI Enhancements
- **Queue Dashboard**: A local web UI to visualize the queue and approve packets without using MCP tools.
